# Контент-план блога «Ставки с головой» (по реальному спросу)

Очередь по частоте Яндекс.Вордстат (Россия, месяц) и просмотрам YouTube. Задача берёт ПЕРВЫЙ slug без `blog/<slug>.html`. `src:` — взять готовый текст из content-src/<файл>; без `src:` — НАПИСАТЬ по брифу.

Каждая статья: 700–1100 слов, спокойно-экспертно, по цифрам, без «железных прогнозов» и гарантий, 18+. Только вечнозелёные объяснялки — никаких прогнозов на конкретные матчи.

## ПЛАНКА КАЧЕСТВА (обязательна)
1. Сильный хук в первом абзаце (конфликт/миф/цифра).
2. Минимум один конкретный числовой пример.
3. Каждая мысль — с «почему».
4. Практический вывод.
5. Никаких выдуманных фактов о реальных командах — только учебные примеры.
6. «Частые ошибки»/«Как применять» + блок ответственной игры и 18+.
7. Заголовки с ключевым словом, без кликбейта.

Формат строки: N. slug | ключевое слово | хук/угол. [src: файл]
